# CoFounder Marketplace — AI Context

## What is this?
A two-sided marketplace for AI agents. Builders create and list agents/teams.
Buyers browse, purchase, and deploy them into their workspace.

## Tech Stack
Next.js 15, React 19, Tailwind, shadcn/ui, tRPC, Drizzle ORM (PostgreSQL),
Clerk (auth), Stripe Connect (payments), Meilisearch (search)

## Revenue Model
1. 20% marketplace take rate (builders keep 80%)
2. 30-40% margin on compute/infrastructure
3. CoFounder Pro agents ($99-499/mo, 100% margin)
4. Platform subscriptions (Free/Pro/Team/Enterprise)
