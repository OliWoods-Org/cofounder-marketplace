# 🏪 CoFounder Agent Marketplace

A two-sided marketplace where builders list AI agents and agent teams, and buyers deploy them with one click. The App Store for AI agents.

## Features

- **Agent Listings** — Browse, search, and deploy individual AI agents
- **Agent Teams** — Coordinated multi-agent departments (3-5 agents working together)
- **Builder Dashboard** — Revenue analytics, leaderboard, payout management
- **Buyer Workspace** — Deploy, monitor, and manage agents with budget controls
- **Leaderboard** — Gamified rankings driving quality and engagement

## Tech Stack

- **Frontend:** Next.js 15 + React + Tailwind + shadcn/ui
- **API:** tRPC (type-safe)
- **Database:** PostgreSQL (Drizzle ORM)
- **Search:** Meilisearch
- **Auth:** Clerk
- **Payments:** Stripe Connect
- **Agent Runtime:** OliBot Nervous System
